Intelligent Resume Screening System 🤖📄
An AI-powered recruitment platform that streamlines the hiring process by automatically analyzing resumes, matching them with job descriptions, and ranking candidates based on relevance and skills.
Built with a focus on clean UI, real-world hiring workflows, and explainable AI, this system mimics how recruiters screen candidates—only faster, smarter, and more consistent.

🚀 Features


📂 Resume Parsing – Upload and extract key details like skills, education, and experience


🎯 Job Description Matching – Compare resumes with job roles using AI-based similarity scoring


📊 Candidate Ranking – Automatically rank applicants based on relevance and fit


🧠 AI Insights – Get human-like explanations for candidate selection or rejection


📉 Skill Gap Analysis – Identify missing skills and improvement areas


📈 Hiring Dashboard – Visualize candidate pipeline and screening analytics


🤝 Candidate Comparison – Compare multiple applicants side-by-side


💬 AI Assistant – Ask questions about candidates and hiring decisions


⚖️ Bias Detection – Promote fair and unbiased hiring recommendations



🎨 UI/UX Highlights


Clean, minimal, recruiter-friendly interface


Light color palette with subtle animations


Real-world hiring workflow visualization


Smooth transitions and interactive dashboards


Resume screening pipeline with animated stages



🛠 Tech Stack


Frontend: React, Tailwind CSS, Framer Motion


Backend: Python (FastAPI / Flask)


AI/ML: Scikit-learn, SpaCy, Transformers


Database: MongoDB / PostgreSQL


Visualization: Chart.js / Recharts



🧩 How It Works


Upload resumes (PDF/DOCX)


Add job description


AI extracts and analyzes candidate data


Calculates match score using NLP models


Ranks candidates automatically


Provides insights and recommendations



🎯 Goal
To build a production-ready AI hiring assistant that enhances recruiter productivity, reduces manual effort, and improves decision-making with intelligent automation.
